# Design Routing Skill

When this skill activates, route based on the design discipline.

## Routing matrix

| User signal | Agent to invoke |
|---|---|
| UI design, component library, visual system, pixel-perfect interface | `design-ui-designer` |
| User research, usability testing, behavior analysis | `design-ux-researcher` |
| Technical CSS architecture, design system implementation foundation | `design-ux-architect` |
| Brand identity, consistency, strategic positioning | `design-brand-guardian` |
| Visual storytelling, multimedia narrative | `design-visual-storyteller` |
| Personality, delight, micro-interactions, easter eggs | `design-whimsy-injector` |
| AI image generation prompt engineering (Midjourney / DALL-E / SD) | `design-image-prompt-engineer` |
| Representative, non-stereotypical AI image generation | `design-inclusive-visuals-specialist` |

## Complementary skills

- General design critique → `design:design-critique`
- Accessibility review → `design:accessibility-review` or `qa-testing-routing` → `accessibility-auditor`
- UX copy → `design:ux-copy`
- Design system spec → `design:design-system` or `ecc:design-system`
- Web design guidelines audit → `web-design-guidelines` skill
- React composition patterns → `vercel-composition-patterns` skill

## When NOT to use

- Frontend implementation (React / Vue components) → `general-engineering-routing` skill or `ecc:frontend-patterns`
- Slide/canvas design → `anthropic-skills:canvas-design`
- Algorithmic / generative art → `anthropic-skills:algorithmic-art`
